# 📚 AI Study Assistant

An AI-powered web application that helps students simplify their study process by summarizing notes, extracting key points, generating exam questions, and explaining content in simple language.

---

## 🚀 Features

- 🧾 Summarize long notes instantly  
- 🔑 Extract key points for quick revision  
- ❓ Generate important exam questions  
- 📖 Explain complex topics in simple language  
- 🎯 Tone selection (Simple / Formal)  
- ⚠️ Smart fallback (works even if API limit is reached)

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit 🌐  
- Gemini API (Google AI) 🤖
