# AI Study Assistant

This project is an AI-powered study assistant that helps students summarize notes, understand homework, and generate exam questions using the Gemini API.

## Features
- Summarize notes
- Extract key points
- Generate exam questions
- Explain in simple language
- Tone selection (Simple / Formal)

## Tech Stack
- Python
- Streamlit
- Gemini API

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   streamlit run app.py

## Use Case
Helps students save time, improve understanding, and prepare better for exams.
